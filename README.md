# 🎬 React Movie App

A modern movie search application built with React + TypeScript + Vite.  
Users can search for movies, view detailed information, and save favorites.

## 🚀 Live Demo
(https://anastasiiaka-com.github.io/react-movie-app/)

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- React Router
- TMDB API
- CSS Modules
- LocalStorage (for favorites & search history)

---

## ✨ Features

- 🔎 Search movies by title
- 🎭 Filter movies by genres
- ❤️ Add/remove movies from favorites
- 📜 Persistent search history
- 🖼 Fallback image for missing posters
- 📱 Responsive design
- ⚡ Fast build with Vite

---

## 📦 Installation

Clone the repository:
git clone https://github.com/AnastasiiaKa-com/react-movie-app.git
cd react-movie-app

Install dependencies:
npm install

Run development server:
npm run dev

Build for production:
npm run build

---

## 🔑 Environment Variables

Create a .env file in the root:
VITE_TMDB_API_KEY=your_api_key_here

You can get a free API key from TMDB.
